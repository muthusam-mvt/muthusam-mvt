<h2><font color="#2DD4BF">🐍 Animated Contribution Grid Snake Setup</font></h2>
<p><font color="#F9FAFB">To render the dynamic contribution snake shown in the footer of this README, set up a GitHub Actions workflow inside your profile repository by following these instructions.</font></p>

<details>
  <summary><b>📖 View Step-by-Step Configuration Guide</b></summary>
  <br />
  
  ### Step 1: Create a profile repository
  Verify you have a public repository named exactly matching your GitHub username: `muthusam-mvt`.
  
  ### Step 2: Add Workflow Directory
  Create the folder directory path inside the `main` branch of your repository:
  ```bash
  .github/workflows/
  ```
  
  ### Step 3: Write Workflow File
  Create a file named `generate-snake.yml` inside that folder and paste the YAML code block below:
  
  ```yaml
  name: Generate Contribution Snake Animation
  
  on:
    schedule:
      # Automatically run this action every 12 hours
      - cron: "0 */12 * * *"
    workflow_dispatch: # Allows manual triggers from the Actions console
    push:
      branches:
        - main
  
  jobs:
    generate:
      permissions: 
        contents: write
      runs-on: ubuntu-latest
      timeout-minutes: 5
      
      steps:
        # Pulls your contribution grid and builds the SVG animations
        - name: Generate github-contribution-grid-snake.svg
          uses: Platane/snk/svg-only@v3
          with:
            github_user_name: ${{ github.repository_owner }}
            outputs: |
              dist/github-contribution-grid-snake.svg
              dist/github-contribution-grid-snake-dark.svg?palette=github-dark
            
        # Pushes the generated SVG files to the output branch
        - name: Push SVGs to the output branch
          uses: crazy-max/ghaction-github-pages@v3.1.0
          with:
            target_branch: output
            build_dir: dist
          env:
            GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
  ```
  
  ### Step 4: Run the Action
  Navigate to your repository's **Actions** tab on GitHub, click the **Generate Contribution Snake Animation** list entry, click the **Run workflow** dropdown button, and trigger the action. Once executed, a branch named `output` containing the SVGs will be created automatically.
</details>
